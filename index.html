About My Journey

My journey as a web developer started with curiosity. Initially, I was fascinated by how websites worked behind the scenes – how a simple click could perform so many actions. This curiosity pushed me to learn the basics of HTML, CSS, and JavaScript, which opened the door to the world of web development. As I explored further, I realized web development is not just about writing code – it’s about creating experiences. Designing responsive layouts, adding interactive features, and optimizing performance gave me the confidence to build real-world projects. Along the way, I also gained knowledge of programming languages like C, Java, and Python, which helped me understand problem-solving and logic building. Combining this with web technologies allowed me to approach development from both the design and engineering perspective.

Currently, I am passionate about combining AI and Web Development to build smarter, user-friendly applications. My focus is on writing clean, efficient code while paying attention to UI/UX design to ensure every project delivers value.
